    基本功能

    • charAt() ---返回字符串的第 n 个字符，如果不在 0~str.length-1之间，则返回一个空字符串。

      • str.charAt(n)

      • var str = "javascript";
str.charAt(2); //'v'
str.charAt(12); //''

    • indexOf() ---返回 substr 在字符串 str 中首次出现的位置,从 start 位置开始查找，如果不存在，则返回 -1。

      start可以是任意整数，默认值为 0。如果 start < 0 则查找整个字符串（如同传进了 0）。如果 start >= str.length，则该方法返回 -1，除非被查找的字符串是一个空字符串，此时返回 str.length.

      • indexOf(substr[,start])

      • var str = "javascript";
str.indexOf('s'); // 1
str.indexOf('s',6); // -1
str.indexOf('',11); // 10
str.indexOf('',8); // 8

    • lastIndexOf() ---返回 substr 在字符串 str 中最后出现的位置,从 start 位置 向前开始查找，如果不存在，则返回 -1。

      • lastIndexOf(substr[,start])

      • var str = "javascript";
str.lastIndexOf('a');//3

    • substring() ---返回从 start 到 end（不包括）之间的字符，start、end均为 非负整数。若结束参数(end)省略，则表示从start位置一直截取到最后。

      • str.substring(start[, end])

      • var str = 'javascript';
console.log(str.substring(1, 4)); //"ava"
console.log(str.substring(1)); // "avascript"
console.log(str.substring(-1)); //"javascript" 传入负值时会视为0

console.log(str)  //"javascript"

    • slice() ---返回从 start 到 end （不包括）之间的字符，可传负值

      • str.slice(start[,end])

      • var str = 'javascript';
console.log(str.slice(1, 4)); //"ava"
console.log(str.slice(1)); // "avascript"
console.log(str.slice(-1)); //"t" 传入负值时会视为从后往前数
console.log(str.slice(1,-1));//avascrip

console.log(str)  //"javascript"

    • substr() ---返回 str 中从指定位置开始到指定长度的子字符串，start可为负值

      • str.slice(start[,end])

      • var str = 'javascript';
console.log(str.substr(1, 4)); //"awas"
console.log(str.substr(1)); // "avascript"
console.log(str.substr(-1)); //"t" 
console.log(str.substr(-1,6));//"t"
console.log(str.substr(1,-1));//""
console.log(str.substr(0,-1));//""
console.log(str)  //"javascript"

    • replace() ---替换 str 的子字符串

      • str.replace(regexp|substr, newSubStr|``function``)

      • var str = "do you love me";
str.replace('love','hate'); // "do you hate me"

    • search() --- 查找 str 与一个正则表达式是否匹配。如果匹配成功，则返回正则表达式在字符串中首次匹配项的索引；否则，返回 -1。如果参数传入的是一个非正则表达式对象，则会使用 new RegExp(obj) 隐式地将其转换为正则表达式对象

      • str.search(regexp)

      • var str = 'I love JavaScript!';
str.search(/java/); // -1
str.search(/Java/); // 7
str.search(/java/i); // 7
str.search('Java'); // 7

    • match() ---返回一个包含匹配结果的数组，如果没有匹配项，则返回 null。如果参数传入的是一个非正则表达式对象，则会使用 new RegExp(obj) 隐式地将其转换为正则表达式对象

      • str.match(regexp)

      • var str = 'Javascript java';
str.match(/Java/); // ["Java"]
str.match(/Java/gi); // ["java", "Java"]
str.match(/ab/g); // null

    • split() ---返回一个数组，分隔符 separator 可以是一个字符串或正则表达式

      • str.split([separator][, limit])

      • var str = "Hello?World!";
str.split(); // ["Hello?World!"]
str.split(''); // ["H", "e", "l", "l", "o", "?", "W", "o", "r", "l", "d", "!"]
str.split('?'); // ["Hello", "World!"]
str.split('',5); // ["H", "e", "l", "l", "o"]

    • trim() ---去除 str 开头和结尾处的空白字符，返回 str 的一个副本，不影响字符串本身的值

      • str.trim()

      • var str = ' abc ';
str.trim(); // 'abc'
console.log(str); // ' abc '

    • toLowerCase() ---将 str 转换为小写，并返回 str 的一个副本，不影响字符串本身的值

      • str.toLowerCase()

      • var str = 'JavaScript';
str.toLowerCase(); // 'javascript'
console.log(str); // 'JavaScript'

    • toUpperCase() --- 将 str 转换为大写，并返回 str 的一个副本，不影响字符串本身的值

      • str.toUpperCase()

      • var str = 'JavaScript';
str.toUpperCase(); // 'JAVASCRIPT'
console.log(str); // 'JavaScript'
