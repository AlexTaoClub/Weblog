    模块

    模块化能隐藏私有的属性和方法，只暴露出公共接口。

    CommonJS规范

    Node.js的模块系统就采用CommonJS规范。CommonJS标准规定：

    • 一个单独的文件就是一个模块
    • 模块内将需要对外暴露的变量放到exports对象里，可以是任意对象，函数，数组等，未放到exports对象里的都是私有的
    • 用require方法加载模块，即读取模块文件获得exports对象

    注意

    > 在Node.js中，模块对外暴露接口时，实际上就是通过module.exports来实现的。
>
> exports只是对module.exports的一个引用。当替换默认对象来暴露接口时，不能使用exports。

    模块属性

    • module.id: 模块的唯一标识
    • module.loaded：标记模块是否加载
    • module.parent: 引用模块的模块

    常用模块

    • path---处理文件与目录的路径

      • path.join
      • path.resolve
      • path.parse
      • path.format
      • path.basename
      • path.extname
      • path.dirname

    • url---URL格式化和反格式化，用于URL解析，处理

      • url.path
      • url.resolve
      • url.format

    • query string---解析和格式化URL中？后面的参数（查询字符串）

      • querystring.escape(str)
      • querystring.parse(str[, sep[, eq[, options]]])
      • querystring.stringify(obj[, sep[, eq[, options]]])
      • querystring.unescape(str)

    • http--网络模块，不提供路由处理，cookie，缓存等调用。主要实现后端web服务器

      • http.createServer([options][, requestListener])
      • http.get(options[, callback])
      • http.request(options[, callback])

    • events--事件模块

      • EventEmitter
      • emitter.on(eventName, listener)
      • emitter.off(eventName, listener)
      • emitter.once(eventName, listener)

    • file system--文件操作，读写目录，读写文件

      • 通用
        • fs.stat()
        • fs.rename(oldPath, newPath, callback)
      • 文件操作
        • fs.mkdir(path[, mode], callback)
        • fs.readdir(path[, options], callback)
        • fs.rmdir(path, callback)
      • 文件操作
        • fs.writeFile(file, data[, options], callback)
        • fs.readFile(path[, options], callback)
        • fs.unlink(path, callback)
        • fs.appendFile
        • ([(path, data[, options], callback)
        • fs.copyFile(src, dest[, flags], callback)
      • fs.Stats 类
        • stats.isDirectory()
        • stats.isFile()

      每一个异步方法对应都有一个同步方法。方法的名字是在后面 + "Sync"。

      fs.writeFile() 对应-->>同步方法 fs.writeFileSync()

    • stream---流（抽象数据结构）

      • [fs.createReadStream(path, options])
      • [fs.createWriteStream(path[, options])
      • 模块zlib
        • zlib.createGzip([options]) 压缩
        • zlib.createGunzip([options]) 解压缩
